## 1) Home page structure
Observed order on Jeff’s home page:

### Section H1 — Hero / initial challenge
- Headline: direct challenge / frustration pattern
- Main hero headline beneath: coaching for a specific kind of person
- Primary CTA button
- Immediate subhead focused on rebuilding self-trust / regaining control

### Section H2 — Problem identification / who this is for
- Intro for target audience
- Bullet list of pain points / symptoms / frustrations
- Social proof embedded nearby
- Additional testimonial quote block

### Section H3 — Empathy / founder identification with the pain
- Headline about feeling frustrated and stuck
- Short founder empathy narrative
- Bullet list of earlier struggles
- CTA

### Section H4 — Educational / framework intro
- “What is coaching?” style section
- Short explainer
- Case-study or example-led intro

### Section H5 — Five-part transformation framework
- Intro line about what high-performing people actually need
- 5 numbered pillars / domains
- Short explanatory paragraph under each
- Bridge line and CTA into flagship program

### Section H6 — Benefit stack / outcomes
- Headline about coaching going beneath the surface
- Repeating bullet / value statements

### Section H7 — Content/news highlights
- Latest posts / article previews
- Category labels
- Short excerpts
- “View all posts” CTA

### Section H8 — Destination cards / next steps
- “Where do you want to go next?” style prompt
- 2–3 cards linking to major site destinations

### Section H9 — Footer
- Short brand line
- footer nav
- socials
